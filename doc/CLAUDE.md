# 南京项目 - 609风洞测控软件系统

## 项目背景
- 甲方：中国航空工业空气动力研究院（南京609项目）
- 乙方：哈尔滨测创科技有限公司
- 建设目标：风洞试验测控系统，含主控软件、数据采集软件、数据库及客户端
- 合同工期：合同签订后3个月内完成安装调试
- 维护期：验收后12个月，8小时响应，2工作日现场

## 技术选型
- 开发语言：C#（主控+数采+数据库），支持C/Python二次开发
- 数据库：MySQL社区版 或 SQLite
- 开发框架：.NET Framework / .NET，WPF
- 通讯协议：TCP/IP、Modbus（RS232/RS485/RJ45/OPC/CAN）
- 硬件：PXI（32路电流卡+32路电压卡+8路天平卡）、64路压力扫描阀

## 软件架构
- **主控软件**：7个子系统通讯（动力/冷却/台架/负载/龙门架/桨距角/安全联锁）、PID风速闭环控制、三级报警22条事件、可视化（流程图/波形/列表）、参数配置、试验管理
- **数据采集**：PXI多源采集+压力阀、工程量转换+滤波、监控显示、TCP数据分发、CSV存储、数据回放
- **数据库客户端**：5张核心表、RBAC权限、关键词+时间查询、表格/曲线展示、Excel/CSV导出、队列安全存储+每日备份

## 核心指标
- 风速：20~120m/s，精度±1%FS，调节<40s
- 台架：俯仰/侧滑 -20~+20°，精度±0.5°
- RAT：推力0~5000N、扭矩0~200N·m、转速0~10000r/min
- 7×24h连续运行，多用户响应≤8秒
- CPU占用率<30%（方案承诺，非甲方要求）
