## 记忆口诀

```
主控：七系通讯PID风，冷却台架龙门架；
      三级报警二十二，流程图+波形图+参数配。

采集：PXI压力多源采，工程量转+滤波；
      列表波形分组看，TCP分发CSV存，回放滤波配置全。

数据库：五张表，关系模型加密存；
        向导建表数据换，RBAC权限搜导出；
        队列存储+日备+一键恢复。

      标准UI三件套，C/Python二开发。
```

## 六、技术指标汇总

| 指标 | 要求 |
|------|------|
| 风速调节范围 | 20m/s ~ 120m/s |
| 风速调节精度 | 不低于 ±1%F.S. |
| 风速调节时间 | RAT释放稳定后 < 40s |
| 试验区温度 | < 55℃ 长时间稳定运行 |
| 俯仰角范围 | -20° ~ +20°，精度 ±0.5° |
| 侧滑角范围 | -20° ~ +20°，精度 ±0.5° |
| 推力测量范围 | 0 ~ 5000N |
| 扭矩测量范围 | 0 ~ 200N·m |
| 转速测量范围 | 0 ~ 10000r/min |
| 桨距角测量范围 | 0° ~ 50° |
| 开发语言 | C++/C#/Python |
| 数据库 | MySQL社区版/SQLite |
| CPU占用率 | < 30% |
| 7×24小时运行 | 客户端软件连续运行能力 |
